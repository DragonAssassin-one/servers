# 安全策略

感谢你帮助保障 Model Context Protocol 及其生态的安全。

## 重要说明

本仓库中的服务器均为 **参考实现**，用于演示 MCP 特性与 SDK 用法。它们面向希望自行构建 MCP 服务器的开发者作为教学示例，**并非**可直接用于生产环境的方案。

本仓库 **不接受** 安全漏洞报告。若在 MCP SDK 中发现漏洞，请在对应 SDK 仓库中报告。

## 在 MCP SDK 中报告安全问题

若在 MCP SDK 中发现安全漏洞，请通过相关 SDK 仓库的
[GitHub Security Advisory 流程](https://docs.github.com/zh/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
私下报告。

请 **不要** 通过公开的 GitHub Issue、Discussion 或 Pull Request 报告安全漏洞。
